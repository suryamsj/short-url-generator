## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/suryamsj/short-url-generator

# Access
$ cd short-url-generator

# Install dependencies
$ npm install

# Run the project
$ npm run dev

# The server will initialize in the <http://http://localhost:5173/>
```
